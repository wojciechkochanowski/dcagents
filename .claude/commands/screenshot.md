## Screenshot Command

When user types **'screenshot'** or asks to check screenshot/screen capture/zrzut ekranu, execute these steps:

### 1. Navigate to `~/Desktop/`
All screenshots saved here
**Always refresh** file list - folder contents may have changed

### 2. Check Latest Screenshot
Find most recent image file - this is the latest screenshot

### 3. Describe What You See
Ensure you see what user intended to show
User will confirm and tell you what to do with this information
